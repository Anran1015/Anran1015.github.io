Science Meets Art: Vuforia + Unity

2018. Nov

An image-based AR project I created for my psychology class: Scan the image to trigger the interactive brain model. It provides a more engaging way for people to understand the functionality of brain and combines artistic project and science education.

I created this project in Unity using Vuforia plugin and imported the modified brain model from Cinema 4D. Using the low-poly shader, I added the artistic and glowing effect to the model.
