Co-Art Cafe, Web AR Project

2019. Nov

A web AR application developed as part of our experience for a one-day pop up cafe project. By scanning the unique marker composed by artists’ initials, the audience is able to quickly access artists’ bio, statement and their portfolio website. It is also adds the interactiveness to the customer experience.

My role in this project is web AR developer. We use A-frame as the development language to create the whole application, along with my skill in HTML/CSS to create a simple app launch page.
